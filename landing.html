Full Stack com Flask e Flet

Aplicação de backend em Flask para gerenciar filmes e frontend em Flet para listar registros e cadastrar novos dados via API.

O que está incluso

  • • Backend em Flask com endpoints REST para filmes, séries, atores e gêneros.
  • • Frontend desktop em Flet consumindo o endpoint `GET /movies`.
  • • Formulário que envia `POST /movies` e mostra feedback ao usuário.
  • • Página estática com instruções de execução usando Tailwind CSS.

Instruções de execução

1. Iniciar o backend

cd apps/backend
python main.py

2. Iniciar o frontend

cd apps/frontend
.\.venv\Scripts\Activate.ps1
python main.py

3. Abrir a landing page

Abra o arquivo landing.html no navegador para ver a apresentação do projeto.

Feito com Flask no backend e Flet no frontend.
